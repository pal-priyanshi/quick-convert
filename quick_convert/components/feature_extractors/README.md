# Feature Extractors

This is for abstractions of the types of feature extractor, e.g.:

* Speaker embedding
* F0

**NOT** `pYAAPT`, `ECAPA-TDNN`, etc. Those should go under their respective categories, (e.g. [speaker encoders](https://github.com/benluks/quick-convert/tree/main/quick_convert/components/speaker_encoders))
